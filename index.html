1. Escopo do Projeto

Objetivo Principal: Criar um site para eventos que permita a inscrição de participantes (acampantes) e voluntários (equipantes), além de um painel de gestão para ambos.

Público-Alvo:

  • Acampantes: Pessoas interessadas em participar do evento.
  • Equipantes: Pessoas interessadas em trabalhar como voluntárias no evento.
  • Administradores: Pessoas responsáveis pela gestão do evento.

Funcionalidades Principais:

  • Página Inicial: Informações gerais sobre o evento, chamada para ação para inscrição.
  • Página de Inscrição para Acampantes: Formulário de inscrição, opção de pagamento, confirmação por e-mail.
  • Página de Inscrição para Equipantes: Formulário de inscrição, informações sobre responsabilidades, confirmação por e-mail.
  • Sistema de Pagamentos:
    • Integração com gateway de pagamento
    • Processamento de pagamentos para acampantes
    • Processamento de pagamentos para equipantes
    • Geração de comprovantes
    • Controle de status de pagamento
    • Relatórios financeiros
  • Painel de Gestão: Login para administradores, listagem de acampantes e equipantes, exportação de dados, envio de e-mails em massa, gestão de informações do evento.
  • Sistema de Autenticação: Login/logout para administradores.
  • Sistema de Notificação e Comunicação: Envio de e-mails (confirmação de inscrição, avisos, etc.).

Requisitos Não Funcionais:

  • Responsividade: Acessível em diferentes dispositivos.
  • Segurança: Proteção dos dados dos usuários.
  • Desempenho: Rápido e eficiente.
  • Usabilidade: Fácil de usar e intuitivo.
  • Segurança dos Dados: Proteção contra acessos não autorizados e vulnerabilidades.
  • Escalabilidade: Capacidade de lidar com um número crescente de usuários e dados.
  • Performance: Bom tempo de resposta e carregamento.

2. Escolha de Tecnologias e Ferramentas

Frontend:

  • HTML, CSS, JavaScript
  • Framework CSS: Bootstrap
  • Framework/Biblioteca JavaScript: React
  • Gerenciamento de Estado: Context API
  • Roteamento: React Router
  • Estilização: CSS Modules
  • Formulários: React Hook Form
  • Requisições HTTP: Axios
  • Testes: Jest e React Testing Library
  • Linter e Formatter: ESLint e Prettier
  • Bundler: Vite
  • Gerenciamento de Pacotes: npm/yarn
  • Component Library: Material UI (opcional)
  • Gateway de Pagamento: Stripe/Mercado Pago

Backend:

  • Linguagem: Node.js
  • Framework: Express.js
  • Banco de Dados: MySQL
  • ORM: Sequelize
  • Autenticação: JWT
  • Validação: Joi
  • Testes: Jest e Supertest
  • API Documentation: Swagger/OpenAPI
  • Logging: Winston
  • Environment Variables: dotenv
  • CORS: cors

Outras Ferramentas:

  • Git, npm/yarn, VS Code, Docker, GitHub Actions, DBeaver
